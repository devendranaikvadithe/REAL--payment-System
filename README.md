# PayWallet – Full Stack Fintech Payment System

## A production-inspired fintech payment platform built with React.js, Node.js, Express.js, MongoDB, and JWT Authentication.

Features include:

- Secure User Authentication
- Email OTP Verification
- Wallet Management
- Bank Account Linking
- KYC Verification
- Transaction Processing
- Transaction History
- Protected APIs using JWT
- Responsive UI
- RESTful Backend Architecture


## Tech Stack

Frontend:
- React.js
- React Router
- Axios
- React Hook Form

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Bcrypt
- Nodemailer

Security:
- JWT Authentication
- Password Hashing
- Rate Limiting
- Helmet Security Headers
